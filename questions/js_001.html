What is closure in JavaScript?

A closure is a function that remembers its outer variables and can access them even after the outer function has executed.

Explanation:

Closures allow for private variables, preserving function states across invocations. This enables data encapsulation and makes functions more powerful in JavaScript.

Code Example:
function outer() {
  let a = 1;
  return function inner() {
    console.log(a);
  };
}
const innerFunc = outer();
innerFunc(); // Output: 1
