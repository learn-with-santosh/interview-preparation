What is data binding in Angular?

Data binding is a mechanism in Angular that allows you to bind the UI and business logic of the application. It is a way to coordinate the communication between the component and the DOM.

Explanation:

There are four types of data binding in Angular:

  • Interpolation: Allows binding of component properties to the DOM.
  • Property Binding: Binds the value of HTML elements to component properties.
  • Event Binding: Allows components to listen and respond to DOM events.
  • Two-way Data Binding: Enables synchronization of data between the model and the view.
