What is a component in Angular?

A component in Angular is the fundamental building block of the user interface. Each component consists of three parts: the HTML template, the TypeScript class, and the CSS styles.

Explanation:

A component in Angular is created using the `@Component` decorator, which defines the component's selector, template, and style files. Components are reusable, and they can be nested inside other components to create complex UIs.

Code Example:

import { Component } from '@angular/core';

@Component({
  selector: 'app-root',
  templateUrl: './app.component.html',
  styleUrls: ['./app.component.css']
})
export class AppComponent {
  title = 'my-angular-app';
}
